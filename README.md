# StoryBeat Board

故事演示板 - 一个基于 iOS 18 设计风格的故事创作工具，帮助创作者可视化故事情节和情绪走向。

## ✨ 功能特性

### 🎨 iOS 18 设计风格
- **灵动岛工具栏** - 顶部胶囊形导航栏，标题可自定义编辑
- **玻璃拟态效果** - 毛玻璃背景，半透明卡片
- **超大圆角设计** - 28px-36px 圆角，符合现代设计趋势

### 🎭 极性情绪系统
- **开头极性** - 卡片左侧，表示故事开始的情绪
- **结尾极性** - 卡片右侧，表示故事结束的情绪
- **顶部大符号** - 自动同步结尾极性，直观显示整张卡片的情绪走向
- **颜色编码**：绿色(+)表示积极，红色(-)表示消极

### 🔗 连贯性检查
- 自动检测相邻卡片的情绪连贯性
- 连贯时显示彩色连接线
- 不连贯时显示⚠️警告提示

### 🎨 6种主题
- 默认、海洋蓝、深色、樱花粉、森林绿、薰衣草

### 📝 核心功能
- **行管理** - 创建、编辑、删除故事行
- **卡片系统** - 支持拖拽排序，跨行移动
- **自动保存** - 数据自动保存到 localStorage
- **导入/导出** - 支持 JSON 格式

## 🛠 技术栈

- HTML5 / CSS3 / JavaScript (原生)
- [SortableJS](https://sortablejs.github.io/Sortable/) - 拖拽排序

## 📖 使用方法

1. 克隆仓库或下载源码
2. 直接在浏览器中打开 `index.html`

### 基本操作

| 操作 | 说明 |
|------|------|
| 添加行 | 点击底部的"+ 添加新行"按钮 |
| 编辑行标题 | 点击行标题直接编辑 |
| 添加卡片 | 点击行内的"+"按钮 |
| 编辑卡片 | 点击卡片进入编辑模式 |
| 删除卡片 | 鼠标悬停卡片，点击右上角的×按钮 |
| 修改页面标题 | 点击顶部导航栏中间的标题 |
| 切换主题 | 点击"外观"按钮选择主题 |

### 极性符号说明

| 符号 | 颜色 | 含义 |
|------|------|------|
| + | 🟢 绿色 | 正面情绪/积极基调 |
| - | 🔴 红色 | 负面情绪/消极基调 |

### 连贯性规则

- **连贯**：上一张卡片的**结尾极性** = 这张卡片的**开头极性**
- **不连贯**：显示⚠️警告，提示需要调整故事节奏
