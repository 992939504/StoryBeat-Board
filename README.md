# StoryBeat Board

故事演示板 - 一个高性能、可持久化的故事创作工具

## 功能特性

- 📝 **行管理** - 创建、编辑、删除故事行
- 🃏 **卡片系统** - 支持拖拽排序的卡片管理
- 🎭 **极性标记** - 三向极性符号（+/-）表示情绪和基调
- 💾 **自动保存** - 数据自动保存到 localStorage
- 📤 **导入/导出** - 支持 JSON 格式的数据导出和导入
- 🎨 **简约设计** - 蓝色背景，白色线条，极致简约风格

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- [SortableJS](https://sortablejs.github.io/Sortable/) - 拖拽排序

## 使用方法

1. 克隆仓库或下载源码
2. 直接在浏览器中打开 `index.html`

### 基本操作

- **添加行**: 点击行下方的 "+ 添加新行" 按钮
- **编辑行标题**: 点击行标题直接编辑
- **添加卡片**: 点击行内的 "+" 按钮
- **编辑卡片**: 点击卡片进入编辑模式
- **删除卡片**: 鼠标悬停卡片，点击右上角的 × 按钮
- **删除行**: 点击行右侧的 "删除行" 按钮
- **拖拽排序**: 拖动卡片可以改变位置，支持跨行拖拽

### 极性符号

点击卡片左侧、右侧、顶部的 +/- 符号可以循环切换：
- `+` 表示正面情绪/基调
- `-` 表示负面情绪/基调

### 数据管理

- **导出数据**: 点击右上角的 "导出JSON" 按钮
- **导入数据**: 点击右上角的 "导入JSON" 按钮
